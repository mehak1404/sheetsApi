### Steps to run
->open terminal and 
    1. Run npm install
    2. Run npm start(make sure nodemon is installed)

### login for the authentication and authorization
-> Paste the following url - http://localhost:3000

### for retrieving the data-
    paste the url - http://localhost:3000/spreadsheet/:spreadsheetId;(for fixed range and sheet names sheet_id_1)
    paste the url - http://localhost:3000/spreadsheet/:spreadsheetId?name="your sheet name";
        Add the following inputs to see the data in your spreadsheet

### for updating the data- 
    paste the url - http://localhost:3000/update
    Enter the input and check the result in your sheet.
